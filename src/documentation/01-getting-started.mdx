# Getting Started

Learn how to get started with Bookemoji, a component library for SvelteKit.

## Installation


```js
npm create bookemoji@latest
```


## Setup

BookEmoji is a set of SvelteKit loaders and svelte components.

#### Prerequisites:

- An already-scaffolded SvelteKit application
- You may want to familiarize yourself with the [Project Structure](project-structure)

---

<h3 id="setup-aliases">Setup Aliases</h3>

Within your `svelte.config.js` add two new aliases. <br />

1. `$bookemoji.config` <br /> The alias path of your config **file**.
2. `$bookemoji.stories` <br /> The alias path of your stories **directory**.

```diff
// svelte.config.js
/**
 * @type {import('@sveltejs/kit').Config}
 */
const config = {
  preprocess: [vitePreprocess()],
  kit: {
    adapter: adapter(),
    paths: {
      base: process.argv.includes("dev") ? "" : process.env.BASE_PATH,
    },
+    alias: {
+      "$bookemoji.config": "src/routes/(design)/books/bookemoji.config.ts",
+      "$bookemoji.stories": "src/routes/(design)/books/stories",
+    },
  },
};

```

<h3 id="create-bookemoji-config">Create bookemoji.config.ts</h3>

This file primarily is to configure the `base` of the design system.

```ts
import type { BookEmojiConfig } from "bookemoji";

export default {
  base: "/books",
} satisfies BookEmojiConfig;
```
